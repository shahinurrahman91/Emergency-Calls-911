How to get access to the analysis and visualization of the dataset called Emergency Calls 911

1. First of all, one has to install Python or anaconda distribution of python  
2. One will have to use the Jupyter Notebook as an editor environment along with Python
3. Install all the libraries through command line 
4. Run jupyter notebook from the command line for starting up the karnel
5. Import all the necessary libraries in jupyter notebook for the analysis and visualization
6. Finally, you will get to see all the necessary analysis and visualization performed from different angles on the dataset named Emergency Calls 911
